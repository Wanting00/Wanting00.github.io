MQTT系列(2)-代码实现

发表于 2021-05-23 分类于 MQTT（消息队列遥测传输协议） 阅读次数：
基于回调函数的MQTT简易实现

客户端功能：

  • Publish 模块
    • 发送命令给broker
    • 命令内容为 1, 2, 3, 4…的升序数字
    • QoS等级可以调整QoS-0, QoS-1, QoS-2
    • 发送时间间隔可调整，分别为(0ms, 10ms, 20ms, 50ms, 100ms, 500ms)
    • 发布主题(topic)：counter/“qos”/“delay”，例： counter/1/100 表示QoS-1，时间间隔100ms

  • Controller 模块
    • 订阅两个topic，分别为“request/qos”“request/delay”
    • 根据收到的命令，调整Publish的功能，主要是更改QoS等级 & 命令发送间隔

应用端功能：

  • Analyser模块
    • 订阅 counter/“qos”/“delay”
    • 接受来自broker的命令，这里主要是升序数字
    • 根据接收的信息，分析网络延迟
